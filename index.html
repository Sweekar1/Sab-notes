Admin Setup

Configure Discord as your storage backend

Discord Webhook URL

Every uploaded PDF is sent to this webhook. Discord hosts the file and returns a public CDN link that the site uses for previews — no other server needed.

Webhook saved
⚠️ Discord CDN links contain an expiry token (usually a few days). For permanent access, set up a bot that refreshes attachment URLs. See step 5 below.

Step-by-step Discord Setup

1

Create your Discord server

Open Discord → click + in the left sidebar → Create My Own → For me and my friends. Name it something like Sab Notes Admin.

2

Create a notes channel

Create a text channel named #all-notes. This is where the webhook will post every upload. You can also make separate channels per faculty: #architecture, #civil, etc.

3

Create the webhook → paste URL here

Open #all-notes → gear icon (Edit Channel) → Integrations → Webhooks → New Webhook. Name it Sab Notes Bot. Copy the Webhook URL and paste it in the field above, then click Save.

4

How uploads work

When a student uploads a PDF, the site POSTs it to your webhook with ?wait=true. Discord replies with a message containing the file's CDN URL. That URL is stored in the browser and used for previewing. You see every upload appear in your Discord channel in real time.

5

(Recommended) Create a bot to refresh URLs

Go to discord.com/developers → New Application → Bot tab → Add Bot. Enable Message Content Intent. Use the OAuth2 URL Generator (scopes: bot, permissions: Read Messages, Read Message History) to invite it to your server. With discord.py or discord.js you can build a command like /refresh-urls that reads messages and returns fresh attachment URLs.

6

Managing notes from Discord

Every Discord message shows the note's title, faculty, year, semester, college, file size, and unique Note ID. To delete a note: delete the Discord message (removes the CDN file) AND remove it from the site using the trash icon on the note card. The Note ID in the embed lets you match them up.

Free hosting tip: Host your bot on Railway, Render, or Replit — all have free tiers sufficient for a notes bot.
